Around the Web

Credit Card Generator

Credit Card Generator is a simple app that lets you generate random valid Card number from either Visa, Amex or Mastercard. Its simplicity is hidden behind generating random numbers and following the law of Luhn's Algorithm. Once the program validate the generated number, you see the result.


      

Words Occurency Counter

Words Occurency Counter is a tool which counts number of occurences of each word in text, then creates a new file and sort them in it. It uses structs to temporary store all of the data. Once the whole text is compared, the sorting functions sorts all of the words and occurrences alphabetically. Program is executed from command line using switches (-i for input file and -o for output file). I strongly recommend using books from wikisources.org in txt format.
